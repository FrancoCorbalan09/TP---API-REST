# TP - API REST | Rick and Morty Explorer
La problemática de la Semana 11 consiste en crear una aplicación web sencilla que realice llamados a la API pública de Rick and Morty (https://rickandmortyapi.com
) y muestre la información obtenida de forma organizada en pantalla.
El objetivo es practicar consumo de APIs REST, renderizado dinámico con JavaScript y publicación en GitHub Pages.
